Add video upload functionality to WooCommerce product galleries, specifically designed for Flatsome theme compatibility.

== Description ==

Flatsome Product Video Gallery is a powerful WordPress plugin that seamlessly integrates video upload functionality into your WooCommerce product galleries. Designed specifically for the Flatsome theme, this plugin allows you to:

* Upload videos directly from the product edit page
* Position videos within your product gallery
* Automatically resize videos to match gallery images using CSS
* Play/pause videos without triggering lightbox popups
* Display video thumbnails in mobile gallery sliders
* Maintain full compatibility with Flatsome's Stack style and Featured - 2 columns layout

== Features ==

* **Easy Video Upload**: Simple drag-and-drop or click-to-upload interface in the product edit page
* **Position Control**: Set the exact position where videos appear in your product gallery
* **Responsive Design**: Fully responsive with mobile-optimized video thumbnails
* **Flatsome Integration**: Seamlessly works with Flatsome's gallery styles and layouts
* **Video Controls**: Custom play/pause functionality without interfering with image lightbox
* **Multiple Formats**: Supports MP4, WebM, and OGG video formats
* **File Size Management**: Configurable maximum file size limits
* **Admin Settings**: Comprehensive settings page for customization

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/flatsome-product-video-gallery` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > Video Gallery to configure plugin settings.
4. Edit any product and use the "Product Videos" meta box in the sidebar to upload videos.

== Frequently Asked Questions ==

= Does this plugin work with other themes? =

This plugin is specifically designed for the Flatsome theme. While it may work with other themes, optimal functionality and styling are guaranteed only with Flatsome.

= What video formats are supported? =

The plugin supports MP4, WebM, and OGG video formats by default. You can configure allowed formats in the plugin settings.

= Can I control the video file size limit? =

Yes, you can set the maximum video file size in the plugin settings under WooCommerce > Video Gallery.

= Will videos show in mobile view? =

Yes, the plugin includes full mobile support with video thumbnails that appear in the mobile gallery slider.
